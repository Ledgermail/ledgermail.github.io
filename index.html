What is LedgerMail

A decentralised platform to send and receive
secure & encrypted emails.

Ledgermail is a blockchain based email service, which is controlled by it's users.

Email is the standard for communication, tragically email services are composed with no privacy and no security. 272 million email records and passwords of email accounts are being offered to be purchased on the Dark Web for under $200.

Ledgermail is built on blockchain technology and is fully decentralized, hence no central authority can control your email accounts. Ledgermail uses asymetric encryption meaning only receiver can read emails with his/her wallet private key, providing transcendent security and privacy.

Asymmetric Encryption

Secure and Private Emails

Ledgermail uses asymmetric encryption using user's wallet address and private key. The xsalsa20 used by ledgermail, is a stream cipher algorithm to encrypt the information which is a fast and secure method used for encrypted communication and storage.

Ledgermail encrypts the email message with receiver's public key (address) inside the browser and sends the message over the network.

The encrypted emails can only be read by the receiver using the ledgermail wallet. The wallet stores the receiver's private key, which is used to decrypt the emails to readable form. Read more in Ledgermail Whitepaper

Ease of Use

Browser extension to access ledgermail

Ledgermail integrates seamlessly with your browser using a chrome extension. The extension works as a wallet to store the ledgermail cryptocurrency. You can also send and receive ledgermail coin using the ledgermail extension.

The ledgermail extension integrates ledgermail with the Gmail Web App. You can view the encrypted emails sent to you without any trouble inside the Gmail Web App. You can send encrypted emails from Gmail interface itself by choosing to send secure email while you type an email.

We are working to bring the ultimate ledgermail experience to all email web applications on all possible browsers in coming updates. Please take a look at our roadmap for more updates.

Power of Blockchain

The Decentralized Ecosystem

The blockchain network is the underlying infrastructure that supports ledgermail service operate in decentralized ecosystem. Ledgermail uses DPoS algorithm which provides faster transactions and less energy consumption.

This decentralized network is in no control of any authority and is supported by ledgermail users. The users can become nodes in the network by registering as a deligate. You can register as a deligate right from the ledgermail wallet to become a node in the network.
